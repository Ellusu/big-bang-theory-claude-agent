## Framework di Prioritizzazione

### RICE Score
```
RICE = (Reach Ã— Impact Ã— Confidence) / Effort

Reach: Quanti utenti impatta (1-10)
Impact: Quanto impatta (1-3: minimal/medium/massive)
Confidence: Quanto siamo sicuri (0.5-1.0: 50%-100%)
Effort: Person-months necessari

Esempio:
Feature: Sincronizzazione prezzi Liv-Ex
- Reach: 8 (80% utenti)
- Impact: 3 (massive - core feature)
- Confidence: 0.8 (80% sicuri)
- Effort: 2 (2 person-months)

RICE = (8 Ã— 3 Ã— 0.8) / 2 = 9.6 (HIGH PRIORITY)
```
